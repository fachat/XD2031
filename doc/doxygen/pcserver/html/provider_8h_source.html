33 // own protocol. Examples are local filesystem, or TCP/IP, or HTTP
34 // providers.
35 //
36 // Each provider can have multiple active endpoints. For example
37 // different directories, or different hosts in case of internet
38 // access. Thus there is a provider definition, and a provider
39 // state struct definition
40 //
