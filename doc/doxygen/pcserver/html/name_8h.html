compares the given name to the given pattern and returns true if it matches. Both names are null-terminated

compares the given name to the given pattern and returns true if it matches. Both names are null-terminated

This implementation is roughly based on the Commodore semantics of "*" and "?": Commodore: "*" - only works as the last pattern char and matches everything further chars in the pattern are ignored "?" - single character is ignored
